# Resume Link Studio · 简历编辑器

纯前端简历编辑工具，支持实时预览、多模板切换、PDF 导出。

## 功能

- 编辑简历基本信息、链接、自定义模块
- 三种模板：经典 / 紧凑 / 现代
- 实时预览
- 导出带超链接的 PDF
- 数据导入 / 导出（JSON）
- 数据自动保存到 localStorage

## 使用方式

### 方式一：本地打开（推荐）

1. 克隆或下载本仓库到本地
   ```bash
   git clone https://github.com/BanananaCavendish/resume-builder.git
   ```
2. 用浏览器直接打开 `index.html` 即可，无需安装任何依赖

### 方式二：通过 HTTP 服务器（PDF 导出需要）

PDF 功能需要联网加载第三方库，某些浏览器对 `file://` 协议有限制，建议用 HTTP 服务器打开：

```bash
# Python 3（内置）
python -m http.server 8080 -d /path/to/resume-builder
# 然后打开 http://localhost:8080

# 或者用 Node.js（需安装 http-server）
npx http-server /path/to/resume-builder -p 8080
# 然后打开 http://localhost:8080
```

## 详细使用步骤

### 1️⃣ 填写基础信息

在左侧面板依次填写：
- **姓名** — 你的中文或英文名
- **目标职位** — 如"前端开发工程师"
- **联系方式** — 城市、电话、邮箱等（自由格式）
- **个人简介** — 一段简短的自我描述

所有内容会在右侧**实时预览**，无需手动刷新。

### 2️⃣ 添加链接

1. 点击 **"添加链接"** 按钮
2. 填写**显示文字**（如"GitHub"、"个人博客"）
3. 填写**超链接地址**（完整 URL 或自动补全）
4. 最多添加 **3 个**链接
5. 可用 **上移/下移** 调整顺序，或点击 **删除** 移除

> 链接支持 GitHub、掘金、知乎、个人网站等常用地址，也会在导出的 PDF 中保存为可点击的超链接。

### 3️⃣ 添加简历模块

1. 点击 **"添加模块"** 按钮
2. 填写**模块标题**（如"工作经历"、"项目经历"、"教育经历"、"专业技能"）
3. 填写**内容**（支持多行文本）

内容中的换行和缩进会保留，适合写项目列表和描述。

### 4️⃣ 切换模板

点击顶部的模板按钮切换样式：

| 模板 | 特点 |
|---|---|
| **经典** | 标准简历布局，信息层级分明 |
| **紧凑** | 间距更紧凑，适合内容较多的简历 |
| **现代** | 侧边栏式布局，个人信息与内容左右分栏 |

切换后预览会立即更新。

### 5️⃣ 导出 PDF

点击 **"导出 PDF"** 按钮（右上角或预览区上方）：

- 自动生成 **A4 尺寸** 的 PDF 文件
- PDF 中的链接为**可点击的超链接**
- 文件名为 `{姓名}-简历.pdf`
- 如果 CDN 加载失败，会自动降级为浏览器打印（`Ctrl+P`）

> ⚠️ PDF 导出需要联网加载 html2canvas 和 jsPDF 库。导出时请确保网络畅通。

### 6️⃣ 导入 / 导出数据

- **导出数据** — 将当前简历保存为 JSON 文件，方便备份
- **导入数据** — 选择之前导出的 JSON 文件恢复简历
- 浏览器本地也会**自动保存**，关闭页面后重新打开数据仍在

### 7️⃣ 恢复示例数据

点击 **"示例"** 按钮，一键填充演示数据，适合初次体验。

## 技术栈

- 原生 HTML / CSS / JavaScript，无构建工具
- [html2canvas](https://html2canvas.hertzen.com/) — 页面截图
- [jsPDF](https://github.com/parallax/jsPDF) — PDF 生成
